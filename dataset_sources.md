## Public dataset sources (beginner-friendly)

Use these sources to replace the virtual simulation dataset with real public data later.

### Temperature (global / land)
- **Berkeley Earth**: Global land temperature data (monthly / yearly). Search: "Berkeley Earth land temperature data"
- **Kaggle**: “Global Land Temperatures” datasets (CSV). Search: "Kaggle GlobalLandTemperaturesByCountry"

### Rainfall / precipitation
- **NOAA Climate Data Online (CDO)**: Station-based daily/monthly precipitation and temperature (requires selecting stations).
- **WorldClim** (climatology): long-term average climate rasters (advanced; more geospatial).

### CO₂ concentration
- **Our World in Data (OWID) CO₂ and greenhouse gas emissions**: country and global time series.

### Sea level
- **NOAA Sea Level Rise** resources or curated sea level datasets on OWID.
- **NASA** sea level explanations and resources (datasets may require extra steps).

### What to store in `data/raw/`
- Keep downloaded CSV(s) in `data/raw/` (do not commit huge files if very large).
- Add a short note in the README about where to download and how to place files locally.

### Citation / ethics note (for your README)
- Cite the dataset source and link.
- Keep the dataset license in mind (most are fine for educational use, but always check).
